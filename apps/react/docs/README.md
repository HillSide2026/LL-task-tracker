# Levine LLP Matter Platform Docs

This Docusaurus app is retained for internal documentation around the Levine LLP matter platform.

Published Levine LLP documentation lives under `docs-live/`.

Most MDX content under `docs/` is inherited upstream platform documentation and is retained only as repository reference material. It is not part of the published Levine LLP docs route.

## Local Development

```sh
yarn start
```

## Build

```sh
yarn build
```
